זמן זהב
לו״ז חדש הלוזי״ם שלי איך משתמשים התחברות

איך משתמשים

ברוכים הבאים לאתר "זמן זהב". כאן תוכלו למצוא מדריכים והסברים כיצד ליצור, לשמור ולנהל את הלוזים שלכם.

שלבים לשימוש:
1. בחרו "לו״ז חדש" ליצירת לוז חדש.
2. בחרו את סוג הלוז (למידה, חופשה, ניהול יום).
3. מלאו את הפרטים, הוסיפו חלקים ללוז ולבסוף שמרו אותו.
4. בדף "הלוזים שלי" תוכלו לצפות, לערוך או למחוק את הלוזים השמורים.

בנוסף, ניתן לצלם מסך של הלוז, להעתיקו להודעה ולשתף אותו בקלות.
